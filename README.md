# Air Modal

## Первый запуск

### 1. Установка библиотек

```sh
bun install

# или

npm install
```

### 2. Запуск проекта

```sh
bun dev

# или

npm run dev
```

## Проверка проекта, сборка и деплой

### Форматирование

```sh
bun format

# или

npm run format
```

### Сборка

```sh
bun run build

# или

npm run build
```

### Деплой на gh-pages

```sh
bun run deploy

# или

npm run deploy
```
